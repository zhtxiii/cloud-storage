# ☁️ 简单云盘 (Cloud Storage)

这是一个基于 Node.js 的轻量级私有云盘应用，支持文件管理、目录上传下载以及拖拽上传等功能。

## ✨ 主要功能

- **📂 文件浏览**：清晰的文件列表视图，支持面包屑导航。
- **📤 文件上传**：
  - 支持多文件同时上传。
  - **支持文件夹上传**（保留目录结构）。
  - **拖拽上传**：支持将文件或文件夹直接拖入浏览器窗口上传。
  - 上传进度条显示。
- **⬇️ 下载支持**：
  - 单文件直接下载。
  - **文件夹打包下载**（自动压缩为 ZIP 格式）。
- **📁 目录管理**：
  - 创建新文件夹。
  - 删除文件或文件夹。
- **🎨 界面友好**：
  - 自动识别文件类型并显示对应图标。
  - 响应式布局。

## 🛠️ 技术栈

- **后端**：Node.js + Express
- **前端**：原生 HTML5 / CSS3 / JavaScript
- **核心依赖**：
  - `multer`: 处理文件上传。
  - `archiver`: 处理目录压缩打包下载。

## 🚀 快速开始

### 1. 环境要求
确保你的系统已安装 [Node.js](https://nodejs.org/) (建议 v14+)。

### 2. 安装依赖
在项目根目录下运行终端命令：

```bash
npm install
```

### 3. 启动服务器
运行以下命令启动服务：

```bash
npm start
```

### 4. 访问应用
打开浏览器访问：[http://localhost:3000](http://localhost:3000)

## 📂 项目结构

```
.
├── server.js           # 后端服务器入口 (API 接口定义)
├── package.json        # 项目配置与依赖
├── public/             # 前端静态资源
│   ├── index.html      # 主页面
│   ├── style.css       # 样式文件
│   └── app.js          # 前端逻辑 (UI交互、API调用)
└── storage/            # (自动创建) 文件存储目录
```

## 📝 常见问题

- **存储位置在哪里？**
  所有上传的文件都会保存在项目根目录下的 `storage` 文件夹中。
- **如何修改端口？**
  编辑 `server.js` 文件中的 `PORT` 变量即可（默认为 3000）。

## ⚠️ 注意事项

- 本项目为演示性质的简易云盘，未包含用户认证（登录/注册）系统。
- 请勿在该环境中存储敏感私密数据，除非你已在这个基础上添加了相应的安全机制。
